Module Specifications

These specification pages will describe the workings of each FOX Module. The following sections will be displayed:

  • A brief description of the module.
  • Libraries Used - A list of library modules that are included, so that their functionality can be used.
  • States - Each state represents a screen that is used in the module.
  • Sub-Modules Called - A list of each module that the current module makes calls to.
  • Entry Themes - A list of points of entry for the module. Each entry theme will be listed with a name, description, and a list of expected input and output parameters. Parameters will be displayed in bold if they are required, and will include an xpath location and the type of element.
